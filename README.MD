- 🔭 fungsi? **project ini untuk memantau kondisi di suatu rungan kapan pun dan di mana pun menggunakan database firebase.**
- 🔗 link **-**
- 💁‍♂️ more info:  **secara lengkap project ini untuk memantau suatu kondisi ruangan, dari jarak jauh. Dan juga bisa mengontrol suatu elektronik dari jarak jauh.
